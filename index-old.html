About UGUI: Defluff

Defluff reorganizes the structure of your PNG file to find better ways of organizing the data, this in turn can sometimes shave a few bytes off the filesize.

UGUI: Defluff created by The Jared Wilcurt
UGUI: Defluff is a GUI for the command line program defluff (v0.3.2) created by Joachim Henke.
UGUI: Defluff icon created by Jevgeni Striganov.
